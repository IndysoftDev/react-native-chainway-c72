# react-native-chainway-c72

React native wrapper for the chainway sdk

## Installation

```sh
npm install react-native-chainway-c72
```

## Usage

```js
import ChainwayC72 from "react-native-chainway-c72";

// ...

const result = await ChainwayC72.multiply(3, 7);
```
